This document provides detailed information about the licensing model for **CoreAura: Conexión Externa** WordPress plugin, explaining how the plugin complies with open source principles while maintaining commercial viability.

1. **GPL v2+ Compliance** - Open source code licensing

## GPL v2+ Compliance ✅

- **Source Code Access**: You can view, modify, and study the source code
- **Personal Use**: You can use the plugin for personal projects under GPL terms
- **Modification Rights**: You can modify the code to suit your needs

When using under GPL v2+ terms, you must:

### Q: Is this plugin truly open source?
**A:** Yes, the source code is licensed under GPL v2+, providing full access and modification rights.

### Q: Why do I need to pay if it's open source?
**A:** The payment covers commercial services (support, updates, maintenance) not provided under GPL. The code remains open source.

### Q: Can I use this for my client's website?
**A:** Yes, with a commercial license. For personal/learning use, GPL v2+ terms apply.
